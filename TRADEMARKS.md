# WhyOps Trademark and Brand Policy

This repository's code is open source under Apache-2.0. The **WhyOps** name, logo, wordmark, and visual brand assets are separate from that code license.

## Allowed

- Factually refer to this project as WhyOps.
- Say that your project is based on or forked from WhyOps.
- Link back to the original repository.

## Not Allowed Without Permission

- Ship your fork under the **WhyOps** name.
- Use the WhyOps logo, wordmark, or brand styling as your product identity.
- Suggest your fork is the official WhyOps project.
- Market a hosted or commercial offering as WhyOps if it is not run by the WhyOps team.

## Forks

If you publish a fork:

- choose your own name
- use your own logo and brand assets
- make the fork relationship clear in your README or about page

Questions about brand usage should go through the project maintainers before release.
